Apa itu Software Defined Networking (SDN)?

SDN memisahkan control plane (keputusan routing) dari data plane (penerusan paket).

  • Control Plane: Ryu Controller - membuat keputusan routing
  • Data Plane: OpenFlow Switches - meneruskan paket
  • Keuntungan: Centralized control, programmable, flexible

Apa itu Ryu?

Ryu adalah framework SDN berbasis komponen dengan dukungan OpenFlow (1.0 - 1.5) dan API Python.

Topologi Mesh

Apa itu Topologi Mesh?

Topologi Mesh adalah konfigurasi jaringan dimana setiap node terhubung langsung ke semua node lainnya. Ini menciptakan graf lengkap dengan redundansi maksimum.

  • Redundansi Tinggi: Banyak jalur antara dua node
  • Toleransi Kesalahan: Jaringan bertahan dari kegagalan tautan
  • Jalur Pendek: Koneksi langsung 1 hop antar node

Kekurangan:

  • Biaya: Membutuhkan banyak kabel/port
  • Kompleksitas: Sulit dikelola skala besar
  • Skalabilitas: Tidak praktis > ~100 node

Algoritma Bellman-Ford

Apa itu Bellman-Ford?

Algoritma Bellman-Ford mencari jalur terpendek dari satu sumber ke semua vertex dalam graf berbobot.

Ditemukan oleh: Richard Bellman dan Lester Ford Jr. (1958-1962)

Keunggulan: Dapat menangani bobot negatif (tidak seperti Dijkstra)

Prinsip Relaksasi

d[v] > d[u] + w(u,v) → d[v] = d[u] + w(u,v)

Jika jalur melalui u ke v lebih pendek, update jarak ke v!

Kuis

1. Berapa jumlah tautan di mesh 10 node?
A. 10
B. 45
C. 90
D. 100
Benar! Rumus: N × (N-1) / 2 = 10 × 9 / 2 = 45
2. Apa tujuan waktu kunci 700 detik?
A. Memperlambat jaringan
B. Memastikan topologi stabil sebelum routing
C. Menghemat CPU
D. Bug dalam kode
Benar! Kunci mencegah kesalahan routing selama konvergensi.
3. Di mesh terhubung lengkap, jalur terpendek s1 ke s50?
A. 49 hop
B. 1 hop (langsung)
C. Bergantung beban
D. Tidak tahu
Benar! Di mesh, semua node terhubung langsung.
4. Apa kepanjangan MST?
A. Minimum Spanning Tree
B. Maximum Speed Transfer
C. Mesh Switch Topology
D. Multi Source Table
Benar! MST mengurangi flooding ARP.
5. Kompleksitas Bellman-Ford semua pasangan?
A. O(V)
B. O(V × E)
C. O(V² × E)
D. O(E²)
Benar! O(V×E) × V = O(V² × E)
